DPT - Delta Parser Timeline

Welcome to the Delta Parser Timeline (DPT) user manual. This guide will help you understand and make the most of DPT's features.

Introduction

DPT is a specialized timeline editor designed for musical composition and parameter manipulation. It provides an intuitive interface for creating and managing musical sequences with precise control over various parameters.

Note: This manual covers version 1.0.0 of DPT.

Getting Started

System Requirements

  • macOS 10.15 or later
  • 4GB RAM minimum
  • 1024x768 minimum screen resolution

First Launch

  1. Launch DPT from your Applications folder
  2. The application will create a default settings file on first run
  3. You'll see an empty timeline with the default number of tracks

Interface Overview

Main Window Components

  • Timeline Ruler: Shows time markers and beat divisions
  • Track Area: Contains multiple tracks for clip arrangement
  • Control Panel: Buttons for common operations
  • Search Bar: Parameter-based clip search

Keyboard Shortcuts

File Operations

Ctrl/Cmd + N New Timeline

Ctrl/Cmd + O Open File

Ctrl/Cmd + S Save

Ctrl/Cmd + Shift + S Save As

Edit Operations

Ctrl/Cmd + T Add New Clip

Delete Delete Selected

Ctrl/Cmd + D Duplicate Clip

Return Rename Selected Clip

View Controls

Ctrl/Cmd + Plus Zoom In

Ctrl/Cmd + Minus Zoom Out

Alt + Arrow Keys Pan Timeline

Timeline Management

Zoom Controls

DPT offers multiple ways to control the timeline zoom:

  • Keyboard shortcuts
  • Mouse wheel with Ctrl/Cmd
  • Pinch gesture on trackpad

Grid System

The timeline uses a beat-based grid system that automatically adjusts based on zoom level:

  • Major grid lines appear at beat intervals
  • Minor grid lines show subdivisions
  • Grid snapping helps precise placement

Working with Clips

Creating Clips

  1. Press Ctrl/Cmd + T or click "Add Item"
  2. Click and drag to position the clip
  3. Double-click to edit parameters

Selection

DPT supports multiple selection methods:

  • Click individual clips
  • Drag selection rectangle (with Ctrl/Cmd)
  • Shift-click for range selection

Parameters Reference

Parameter Description Type
cAttacco Starting time of the clip Float
durataArmonica Harmonic duration Float
ritmo Rhythm values List[Float]
durata Duration of the clip Float or List
ampiezza Amplitude values Float or List
frequenza Frequency parameters List[Float]
posizione Position parameter Float/String/List

File Management

YAML Format

DPT uses YAML files to store timeline data. Here's an example structure:

comportamenti:
  - cAttacco: 0.0
    durataArmonica: 35.0
    ritmo: [3.0]
    durata: 10.0
    ampiezza: -14.0
    frequenza: [2.0, 5.0]
    posizione: [-2.0, GEN06]
Important: Always save your work regularly. The application does not currently support auto-save.

Search Functions

Parameter Search

The search bar allows you to find clips based on parameter values:

  1. Select the parameter to search
  2. Enter the search value
  3. Results will be highlighted on the timeline
Tip: You can search for exact values or use lists for complex parameters.

Tips and Tricks

Efficiency Tips

  • Use keyboard shortcuts for common operations
  • Hold Alt while dragging for fine positioning
  • Double-click empty track space to add a clip
  • Use the search function to quickly locate specific parameters

Best Practices

  • Organize clips into logical track groupings
  • Use meaningful names for clips
  • Save work regularly
  • Keep track of parameter ranges for consistency
